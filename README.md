# 🧩 Captcha Solver

A deep learning-based project for solving CAPTCHAs, built with Python, Keras, and OpenCV.

## 🚀 Overview
This project uses deep learning and computer vision techniques to automatically recognize and solve CAPTCHAs. It is designed to demonstrate how to process image data and train a Convolutional Neural Network (CNN) in a Python environment.

## 🛠 Technologies
- **Python**: The core programming language.
- **Keras/TensorFlow**: For building and training the deep learning model.
- **OpenCV**: For image preprocessing (noise reduction and segmentation).
- **NumPy**: For numerical data processing.
- **Scikit-learn**: For data management and model evaluation.

## 📦 Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/nasimrzaii/Captcha-Solver.git
   ```
2. Enter the project directory:
   ```bash
   cd Captcha-Solver
   ```
3. Install the dependencies:
   ```bash
   pip install numpy opencv-python keras tensorflow scikit-learn
   ```

## ⚙️ Usage
To start the solver, run the main script:
```bash
python main.py
```
*(Make sure your dataset or target image is in the correct directory as defined in the script.)*

## 📈 Key Features
- [x] Image preprocessing with OpenCV to enhance model accuracy.
- [x] CNN architecture for robust character recognition.
- [x] Clean and modular code structure.
